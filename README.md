# Nuxt Eslint Config

## Installation

```bash
npm install @favorodera/nuxt-eslint-config
```

## Usage

```mjs
// eslint.config.mjs

import withNuxt from './.nuxt/eslint.config.mjs'
import config from '@favorodera/nuxt-eslint-config'

export default withNuxt(config, {
  // Other ESLint configs
})
```
